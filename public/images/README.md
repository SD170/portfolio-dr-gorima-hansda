# Gallery Images

Place all your photos in this folder.

## Naming Convention

Name your images as:
- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`
- etc.

Or you can use any names you prefer, just update the image paths in `src/components/Gallery.js`

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Recommended Image Sizes

For best quality:
- Main carousel images: 1920x1080px (16:9 aspect ratio)
- Thumbnail images: 800x800px (square)

The gallery will automatically resize and crop images to fit.
